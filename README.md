## 📌 Overview

**Gen-AI Data Analyst System** is an advanced AI-powered analytics platform designed to automate exploratory data analysis and intelligent preprocessing.

The system helps users upload datasets, understand data quality issues, analyze distributions, detect missing values and outliers, check multicollinearity, and prepare data for machine learning workflows.

---

## ✨ Key Features

### 📁 Smart Data Upload
- CSV upload with automatic encoding detection
- UTF-8 and Latin1 fallback support

### 🧹 Data Cleaning
- Drop unwanted columns
- Change column data types
- Remove duplicate rows

### 🎯 Target Detection
- Auto-detects classification and regression problems
- Suggests target columns using heuristics

### 📊 EDA Visualizations
- Histograms with KDE curves
- Mean / Median / Mode markers
- Box plots for outlier inspection
- Categorical count plots
- Correlation heatmaps
- Multicollinearity summary

### ❓ Missing Value Handling
- Mean imputation
- Median imputation
- Mode imputation
- Column dropping for excessive missing values

### 🚨 Outlier Treatment
- IQR-based detection
- Log transformation
- Intelligent binning
- Before/after boxplot analysis
